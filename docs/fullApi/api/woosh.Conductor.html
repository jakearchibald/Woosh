addListener

Add an object to listen for conductor events. Conductor fires the following:

Synopsis

myConductor.addListener(listener, thisVal)

Description

listener.ready() - When the conductor is able to start

listener.start() - When the conductor starts

listener.testSetComplete(testName, resultComparison) - Called when the same test name is completed in each woosh._TestFrame - testName is the name of the test - resultComparison is the woosh.ResultComparison used to run the tests

listener.testComplete(libraryName, testName, results) - Called when a test completes for a particular library - libraryName is the name of the library the test was completed for - testName is the name of the test - results is an instance of woosh.Results. Will be udefined if the test was missing.

listener.allTestsComplete(libraryResults, resultComparisons) - Called when all queued tests have completed - libraryResults is an object of woosh.LibraryResult keyed by library name - resultComparisons is an object of woosh.ResultComparisons keyed by test name
