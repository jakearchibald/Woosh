Description

Test instances are created within a call to woosh.addTests.

Constructor

Synopsis

var myTest = new woosh.Test(loopCount, test)

Parameters

loopCount
Type
number
Required
Yes

Number of times to run the test.

Tests that run longer have less margin of error.

test
Type
Function
Required
Yes

The test to run.

The following params will be passed into the test function:

test
The instance of woosh.Test, provided so you can easily call instance methods on it.

Examples

woosh.addTests('glow-170', {
    'Test Name': new woosh.Test(1000, function(test) {
        // do stuff
        
        // return a value (this will be checked against the results of other tests)
        return returnVal;
    })
});
woosh.addTests('jq-140', {
    'Appending elms': new woosh.Test(1000, function(test) {
        $('<strong/>').appendTo(document.body);
        
        // restrict the element counting to the last loop
        if (test.lastLoop) {
            return $('strong').length;
        }
    })
});

Instance Properties

lastLoop

True if the test is running its last loop

Type
number

Synopsis

myTest.lastLoop

Description

You can use this to restrict calculating the return value to the last loop.

Instance Methods

setResult

Manually set the result of the test.

Synopsis

myTest.setResult(result, unit, highestIsBest)

Description

By default the result is the time the test took to run in milliseconds. However, you may want your test to measure something else like frames-per-second. You can achieve that using this method.

Returns

woosh.Test

Parameters

result
Type
number
Required
Yes

The result value as a number

unit
Type
string
Required
No
Default
'ms'

The unit for the result

highestIsBest
Type
boolean
Required
No
Default
false

Treat high numbers as better than low numbers?

Examples

// using an AsyncTest to measure framerate
woosh.addTests('glow-170', {
    'Test Name': new woosh.AsyncTest(1, function(test) {
        var framesRendered = 0;
    
        var anim = new glow.anim.Animation(3, {
            onFrame: function() {
                framesRendered++;
            },
            onComplete: function() {
                // set the frames per second as the result
                test.setResult(framesRendered/3, 'fps', true);
                test.endTest();
            }
        });
        
        anim.start();
    })
});
